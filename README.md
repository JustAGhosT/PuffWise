# Puffwise

Track. Reduce. Upcycle. Challenge Yourself.

A privacy-first, browser-based nicotine usage tracking platform that empowers users to track, analyze, and reduce all forms of nicotine use—including vapes, cigarettes, and pouches—with unified multi-product logging, challenge-driven harm reduction analytics, milestone celebrations, and streak tracking.

## Documentation

- [Product Requirements Document (PRD)](docs/01_product/01_product_requirements.md)

## Features

- Multi-product nicotine usage tracking (vapes, cigarettes, pouches)
- Challenge-driven reduction goals with streak tracking
- Stepwise device disassembly guides
- Upcycling project gallery
- Collection tracking for community impact
- Full offline support with local storage
- CSV data export
